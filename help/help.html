FORM MAIL_XH – Anleitung

Mit dem Plugin lassen sich in einfacher Weise Formulare erstellen, die an eine voreingestellte Mailadresse versandt werden können.

Formular definieren

Die Struktur der Formulare werden in einer einfachen INI-Datei definiert. Es stehen unterschiedliche Formate zur Verfügung, um auch ein komplexes Formular gestalten zu können.

Format

Die Formular-Definition muss dem INI-Datei-Format mit Gruppen und Key-Value Paaren entsprechen. Die Gruppen in Eckigen Klammern enthalten einen Textstring. Das Key-Value Paar format = ... bestimmt das Anzeigeformat:

  • format = header: Der Textstring wird als Überschrift angezeigt
  • format = text: Der Textstring wird als Absatz angezeigt

Formular aufrufen

Als query kann entweder ein valider MySQL-String übergeben werden, oder der Name einer gespeicherten Query. Die gespeicherten Abfragen sind im Verzeichnig \queries\ des Plugins abgelegt.
